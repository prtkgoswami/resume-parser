# ATS Resume Parser

A local-first tool to inspect how ATS systems extract and parse resume data.

## Tech Stack
- Frontend: React + TypeScript + TailwindCSS
- Backend: Node.js + Express
- Parsing: pdfjs-dist, mammoth
- Deployment: Cloudflare

## Goals
- Visualize ATS-extracted text
- Detect parsing issues (ligatures, glyphs)
- Help candidates fix resumes for ATS
